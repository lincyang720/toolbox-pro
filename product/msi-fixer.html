MSI 修复工具

双击 .msi 安装包没有反应，或提示"没有关联的程序"？ 这是 Windows 中 MSI 文件关联损坏的常见问题，Win7/8/10/11 均可能出现。 本工具一键重新注册 Windows Installer 服务和文件关联， 3 秒内彻底修复，无需重装系统。

  • 重新注册 Windows Installer 服务（msiserver）
  • 修复 .msi 文件注册表关联（HKCR\.msi）
  • 恢复右键菜单「安装」「修复」「卸载」选项
  • 启动 msiserver 服务并设置正确启动类型
  • 刷新系统图标缓存
  • 自动请求管理员权限，无需手动操作

常见问题

修复后 .msi 还是打不开怎么办？

重启电脑后再试，90% 的情况重启后即可恢复。如仍有问题请联系客服协助处理。

支持哪些 Windows 版本？

支持 Win7 / Win8 / Win10 / Win11，凡支持 PowerShell 的 Windows 系统均可运行。

会修改哪些系统设置？

只修复 .msi 相关的注册表项（HKCR\.msi、HKCR\Msi.Package）和 msiserver 服务，不影响其他任何系统设置。

为什么需要管理员权限？

修改 HKCR 注册表项和操作系统服务需要管理员权限，这是 Windows 安全机制要求，与本工具无关。
