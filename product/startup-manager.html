常见问题

禁用启动项会不会导致软件不能用？

不会。禁用开机启动只是让程序不在开机时自动运行，你仍然可以手动双击打开这些软件，功能完全不受影响。

禁用后如何恢复？

在工具中选中已禁用的项目，点击「启用选中」即可恢复，数据不会丢失。

支持哪些 Windows 版本？

支持 Win7 / Win8 / Win10 / Win11，所有通过注册表 Run 键管理启动项的 Windows 系统均可运行。

红色「高影响」的项目能禁用吗？

不建议。高影响通常是显卡驱动、安全软件、输入法等系统关键组件，禁用可能导致硬件无法正常工作。工具在禁用高影响项时会弹出二次确认提醒。
