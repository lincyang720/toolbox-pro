常见问题

修复后还是断网怎么办？

请先重启电脑，90% 的情况重启后即可恢复。若仍有问题，可能是路由器或运营商问题，建议检查路由器状态或拨打运营商客服。

支持哪些 Windows 版本？

支持 Win7 / Win8 / Win10 / Win11，凡支持 PowerShell 和 netsh 命令的 Windows 系统均可运行。

会影响正常网络设置吗？

本工具只重置协议栈和缓存，不修改 Wi-Fi 密码、路由配置等用户设置。重置代理设置仅针对 WinHTTP 代理，不影响浏览器代理插件。

为什么需要管理员权限？

netsh 和 arp 等命令修改系统网络栈需要管理员权限，这是 Windows 安全机制要求，与本工具无关。
